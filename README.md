# ferrous

A structured Rust learning repository — concepts, experiments, and projects built from the ground up.

## Structure

```
ferrous/
├── src/          # active code — current concept being explored
└── projects/     # standalone projects built along the way
```

## Progress

- [x] Cargo, Hello World
- [x] Variables, Mutability, Constants, Shadowing
- [ ] Data Types
- [ ] Functions & Control Flow
- [ ] Ownership
- [ ] Borrowing & References
- [ ] Structs, Enums, Pattern Matching
- [ ] Error Handling
- [ ] Traits & Generics
- [ ] CLI tools, Web backend, WebAssembly

## Setup

```bash
git clone <repo-url>
cd ferrous
cargo run
```
